Extension Blocking Test

This page tests if the extension is blocking sites correctly.

Test Links:

  • YouTube (should be blocked)
  • Google (should work)
  • Facebook (should work)

Instructions:

  1. Make sure the extension is loaded and YouTube is in the blacklist
  2. Click on the YouTube link above
  3. You should see a Chrome blocking page instead of YouTube
  4. Other links should work normally

Debug Information:

Open Chrome DevTools Console and look for:

  • DNR rules being created
  • Blocking messages
  • Extension debug logs
